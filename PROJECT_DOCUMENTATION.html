NLUI-Agentic-AI Project Documentation

NLUI-Agentic-AI is a modern, modular healthcare administration panel built with Next.js, Ant Design, and Tailwind CSS. It supports complex workflows for managing members, claims, providers, pharmacies, authorizations, billing, and more.

Project Setup

  1. Clone the repository and install dependencies:
    npm install
  2. Start the development server:
    npm run dev
  3. Build for production:
    npm run build

Key Features

  • Role-based access and authentication
  • Modular, deeply nested routing for complex healthcare workflows
  • Reusable Ant Design UI components
  • Consistent navigation with Breadcrumbs
  • Responsive, modern design with Tailwind CSS

Complex Pages & Routes
